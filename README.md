
# Code Translator

Paste in code, get it explained three ways: for an engineer, a product
manager, or an executive.

I've spent my career translating between technical and non-technical
people. This tool is that skill, productized. Built with Python (Flask),
JavaScript, and the Anthropic API, with AI pair-programming used openly
and deliberately.

In development. Follow the pull requests to watch it come together.
